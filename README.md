# Healthcare Exploratory Data Analysis (EDA)

## Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a healthcare dataset to uncover patterns, trends, and insights related to patient admissions and outcomes.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis Performed
- Data cleaning and preprocessing
- Handling missing values
- Duplicate detection and removal
- Descriptive statistics
- Visual analysis of key variables
- Insight generation from healthcare data
